### 说明：

在启动后，会每隔1分钟将官方源下载到本地。

当访问本地的11451端口后，会将本地的官方源Json文件输出，从而达到分发官方源的作用


### 使用方法：

```javascript

// 从github上下载包
git clone https://github.com/BSTluo/koishi-search-endpoint.git
/// 进入文件夹
cd koishi-search-endpoint
// 下载依赖包
npm i
// 运行它！
node index.js

```
